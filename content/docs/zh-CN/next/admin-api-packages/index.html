管理Package

重要

此页面仅显示 部分常用操作。

  • 关于pulsar-admin最新最完整的信息，包括命令、参数、描述等等，可参阅pulsar-admin 文档。

  • 关于 REST API 最新最完整的信息，包括参数、响应、示例等等，可参阅 REST API 文档。

  • 有关 Java admin API 的最新完整信息，包括类、方法、描述等，请参阅 Java admin API 文档。

包管理器或包管理系统以一致的方式自动管理包。 这些工具简化了用户的安装任务、升级过程和删除操作。 包是包管理器处理的最小单元。 在Pulsar中，包在租户和命名空间级别进行组织，以管理Pulsar功能和Pulsar IO连接器（即源和汇）。

什么是包裹？

包是用户希望在以后的操作中重用的一组元素。 在Pulsar中，包可以是一组函数、源和汇。 您可以根据需要定义包。

Pulsar中的软件包管理系统存储每个软件包的数据和元数据（如下表所示），并跟踪软件包版本。

元数据说明
描述Package 的描述
联系方式Package 的联系方式： 例如，一个开发团队的email地址。
创建时间Package 的创建时间。
修改时间Package 的最近修改时间
properties一个用户自定义用于存储其他信息的key/value映射表。

如何使用package？

Packages可以有效使用相同的函数和IO连接器组。 例如，你可以在多个命名空间中使用相同的函数，输入源和输出。 主要步骤如下：

  1. 通过提供以下信息在包管理器中创建包：类型、租户、命名空间、包名称和版本。

    组件说明
    type指定支持的包类型之一：函数、接收器和源。
    租户指定要在其中创建包的租户。
    命名空间指定要在其中创建包的命名空间。
    name使用格式指定包的完整名称 <tenant>/<namespace>/<package name>.
    version使用格式指定包的版本 MajorVerion.MinorVersion用数字表示。

    您提供的信息将以以下格式为包创建URL： <type>://<tenant>/<namespace>/<package name>/<version>.

  2. 将元素上载到包中，即要跨名称空间使用的函数、源和接收器。

  3. 从各种名称空间对此包应用权限。

现在，您可以通过在包管理器中调用此包来使用在包中定义的元素。 包管理器通过URL定位它。 For example,

sink://public/default/mysql-sink@1.0
function://my-tenant/my-ns/my-function@0.1
source://my-tenant/my-ns/mysql-cdc-source@2.3

Package中的包管理

您可以使用命令行工具、RESTAPI或Java客户端来管理Pulsar中的包资源。 更具体地说，您可以使用这些工具 上传， 下载和 删除 一个软件包 获取元数据 和 更新包的元数据 获取软件包的版本 和 获取在命名空间下特定类型的所有软件包。

上传 Package

可以使用以下命令上载包。

pulsar-admin
REST API
JAVA
bin/pulsar-admin packages upload function://public/default/example@v0.1 --path package-file --description package-description

POST /admin/v3/packages/:type/:tenant/:namespace/:packageName/:version

使用同步调用的方式上传到 Package 管理服务中。

  void upload(PackageMetadata metadata, String packageName, String path) throws PulsarAdminException;

使用异步调用的方式上传到 Package 管理服务中。

  CompletableFuture<Void> uploadAsync(PackageMetadata metadata, String packageName, String path);

下载 Package

您可以使用以下命令下载软件包。

pulsar-admin
REST API
JAVA
bin/pulsar-admin packages download function://public/default/example@v0.1 --path package-file

GET /admin/v3/packages/:type/:tenant/:namespace/:packageName/:version

从包管理服务同步下载包。

  void download(String packageName, String path) throws PulsarAdminException;

从包管理服务异步下载包。

  CompletableFuture<Void> downloadAsync(String packageName, String path);

删除包

可以使用以下命令删除包。

pulsar-admin
REST API
JAVA

以下命令删除版本为0.1的包。

bin/pulsar-admin packages delete functions://public/default/example@v0.1

DELETE /admin/v3/packages/:type/:tenant/:namespace/:packageName/:version

使用同步调用的方式删除特定版本的 Package。

  void delete(String packageName) throws PulsarAdminException;

使用异步调用的方式删除特定版本的 Package。

  CompletableFuture<Void> deleteAsync(String packageName);

获取 Package 的元数据

可以使用以下命令获取包的元日期。

pulsar-admin
REST API
JAVA
bin/pulsar-admin packages get-metadata function://public/default/test@v1

GET /admin/v3/packages/:type/:tenant/:namespace/:packageName/:version/metadata

使用同步调用的方式获取 Package 的元数据。

  PackageMetadata getMetadata(String packageName) throws PulsarAdminException;

使用异步调用的方式获取 Package 的元数据。

  CompletableFuture<PackageMetadata> getMetadataAsync(String packageName);

更新 Package 的元数据

可以使用以下命令更新包的元数据。

pulsar-admin
REST API
JAVA
bin/pulsar-admin packages update-metadata function://public/default/example@v0.1 --description update-description

PUT /admin/v3/packages/:type/:tenant/:namespace/:packageName/:version/metadata

同步更新包的元数据。

  void updateMetadata(String packageName, PackageMetadata metadata) throws PulsarAdminException;

异步更新包的元数据。

  CompletableFuture<Void> updateMetadataAsync(String packageName, PackageMetadata metadata);

列出 Package 的所有版本

可以使用以下命令列出包的所有版本。

pulsar-admin
REST API
JAVA
bin/pulsar-admin packages list-versions type://tenant/namespace/packageName

GET /admin/v3/packages/:type/:tenant/:namespace/:packageName

使用同步调用的方式列出 Package 的所有版本。

  List<String> listPackageVersions(String packageName) throws PulsarAdminException;

使用异步调用的方式列出 Package 的所有版本。

  CompletableFuture<List<String>> listPackageVersionsAsync(String packageName);

在命名空间下列出特定类型的所有包

可以使用以下命令列出命名空间下特定类型的所有包。

pulsar-admin
REST API
JAVA
bin/pulsar-admin packages list --type function public/default

PUT /admin/v3/packages/:type/:tenant/:namespace

同步列出命名空间下特定类型的所有包。

  List<String> listPackages(String type, String namespace) throws PulsarAdminException;

异步列出命名空间下特定类型的所有包。

  CompletableFuture<List<String>> listPackagesAsync(String type, String namespace);
